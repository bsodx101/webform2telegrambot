# WebForm2TelegramBot

A simple bot to **send form submissions (text + photos)** from a web form directly to a **Telegram chat or channel**.

This repository includes:
- A static HTML form
- A backend script (`telegram.php`) to forward form data to Telegram
- A thank-you page after submission

Perfect for contact forms, feedback forms, or lightweight lead capture forms that need Telegram alerts.
